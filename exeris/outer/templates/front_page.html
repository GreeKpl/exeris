Links

If you are interested in reading more about the game consider visiting:
  • Exeris Blog - articles regarding the plans, game mechanics etc.
  • Taiga.io - the dev wiki and work status
  • Exeris forum (you can log in using the game account created on users.exeris.org)
  • users.exeris.org - the main game instance handling the user accounts. Registration there is required to get access to the forum or the development server.
  • dev.exeris.org - the development server with ability to create test characters. It presents the newest features. Accounts on dev.exeris.org are copied from users.exeris.org, because the data on dev instance is often wiped.
  • GitHub - code repository
